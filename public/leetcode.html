1. Even or Odd

Easy

Given an integer num, return "Even" if the number is even, or "Odd" if the number is odd.

A number is considered even if it is divisible by 2 with no remainder. A number is considered odd if it is not divisible by 2.

Note: Zero (0) is considered an even number.

Example 1:

Input: num = 2

Output: "Even"

Explanation: 2 is divisible by 2, so it's even.

Example 2:

Input: num = 3

Output: "Odd"

Explanation: 3 is not divisible by 2, so it's odd.

Example 3:

Input: num = 0

Output: "Even"

Explanation: 0 is divisible by 2, so it's even.

Example 4:

Input: num = -4

Output: "Even"

Explanation: -4 is divisible by 2, so it's even.

Constraints:

  • -109 ≤ num ≤ 109
  • num is an integer
